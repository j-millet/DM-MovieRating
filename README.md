# Requirements
- python
- jupyter notebook
- dvc (with dvc-gdrive)

# How to run
1. Download the project
2. Run `pip install -r requirements.txt` (admin may be required)
3. Get data (run `dvc pull` in DM-MovieRating folder)
4. Open _code.ipynb_
5. Run cells and enjoy!
